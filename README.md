# app-reviews-scraper

`brew install virtualenv protobuf`


1. App details
1. App reviews
1. Similar apps

Given one app id:

|               | Apple Store | Play Store |
| ------------- | ----------- | ---------- |
| (All) Reviews | Yes         | Yes        |
| App details   | Yes         | Yes        |
| Similar apps  | Yes         | Yes        |

App list:

```json
{
  "project": "project-name",
  "apps": [
    {
      "android": "",
      "ios": ""
    }
  ]
}
```
